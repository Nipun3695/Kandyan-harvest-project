# 🌿 Hill Country Innovators - Sri Lanka Supply Chain

**Hill Country Innovators** is a web-based ecosystem designed to bridge the gap between Sri Lankan farmers and supermarkets through an efficient, real-time supply chain and logistics network.

## 🚀 Key Features

* **Farmer Dashboard**: Add harvest details with descriptions, manage stock, and view market analytics.
* **Supermarket Marketplace**: Search for fresh produce, place orders with offered prices, and track deliveries.
* **Logistics Network**: Dedicated driver panel to accept delivery hires, navigate routes, and update real-time delivery status.
* **Real-time Notifications**: Instant alerts for order placements, acceptances, and delivery milestones powered by Socket.io.
* **User Roles**: Specialized portals for Farmers, Supermarkets, and Drivers.

## 🛠️ Tech Stack

* **Frontend**: HTML5, CSS3 (Custom styles), JavaScript (ES6+)
* **Backend**: Node.js, Express.js
* **Database**: MongoDB (Atlas)
* **Real-time**: Socket.io
* **Maps/Nav**: Google Maps API integration

## ⚙️ Installation & Setup

1. **Clone the repository**:
   `git clone https://github.com/your-username/hill-country-innovators.git`
2. **Install dependencies**:
   `npm install`
3. **Setup Environment Variables**:
   Create a `.env` file and add your `MONGO_URI` and `PORT`.
4. **Run the server**:
   `npm start`
